# Minecraft Server Management

Combines:

- [Velocity Minecraft Proxy](https://gate.minekube.com/)
- [Crafty Controller](https://craftycontrol.com/)
- [Copyparty](https://copyparty.eu/)

## Assumptions
Currently, assumes the stack lives on `/srv/`, and is a Btrfs subvolume.

# Limitations
Crossplay is not enabled yet.

There are permission issues to iron out with adding files to crafty
subdirectories.
